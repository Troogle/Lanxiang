比赛概要：

  1. 本次比赛是面向全体华人的一次OSU Catch the Beat 模式比赛

  2. 比赛主体采用积分淘汰混合制度。在举办方指定的12段时间内参加比赛获取积分，时间截止后选取8位分数最多的选手进入淘汰赛。

  3. 报名方式：4选1

    • 在官网自助报名 推荐
    • 在本帖中留下OSU ID
    • PM举办方裁判或者举办者
    • 入群： 129614174 进行报名，报上OSU ID并@群主

  4. 报名时间：即日起到2月16日24:00

  5. 报名结束后将由裁判团进行审核登记并在OSU论坛帖和官网更新。如果比赛中出现与报名名单中不相符的id，一律不做分数登记，视为无效。

  6. 比赛奖品：

    • 第一名： 150元或等值撒泼特(一年)
    • 第二名： 100元或等值撒泼特(6个月)
    • 第三名： 50元或等值撒泼特(2个月)
    • 第四名： 20元或等值撒泼特(1个月)

赛制流程：

赛季阶段：

  1. 比赛模式：积分制，每天比赛按照前6场的积分计入总分，相同地图按照第一次遇到此图时的积分计入总分，一共有三轮积分赛，每人在赛季阶段最多能获得6*3=18张图的积分

    每天有上午(10:00)，下午(3:00)，晚上(8:00)，午夜(0:00)四场场次
    每天的图池不同，当天的图池内有none*4,hd*3,dt*3,hr*3,free mod*3共16张图
    图池将在报名结束后发布

  2. 积分计算：

    Point = 选手分数÷Auto在none模式下可以获得的分数

    如果中途或结束有fail的情况，选手分数做折半处理

    如果选手出现掉线的情况，分数按照结束时ranking界面显示的分数为准，不接受任何重赛请求
    若担心有突发情况，请在ready前离场，每局一旦开始即视为你可以无干扰完成此图

    上午打满6场的人，下午再打是不会获得积分的，请大家对照自己的网络状况选择对自己有利的场次

  3. 比赛房间：裁判将提前15分钟建立16人Head to Head模式的房间（胜利条件为Score），请PM当场裁判获得房间邀请链接，先到先得，准时开赛，可中途加入比赛

    每场6局，none*2，hd*1，dt*1，hr*1，free mod*1，裁判将依照此模式顺序roll点随机选图

  4. 赛季结束后将选取前8名进入淘汰赛

  5. 选手不一定需要截图，但是可以自行截图作为证据避免裁判团统计失误的情况

淘汰赛阶段：

  1. 分组：原则上按照赛季中积分排名第1,4,5,8和积分排名第2,3,6,7分为两组

  2. 比赛模式：BO9，9局5胜，在队长之间轮流选图，淘汰一组，取胜者组四人进入决赛

  3. 时间：将在积分赛结束后公布

  4. 比赛房间：裁判将提前15分钟建立Team VS模式的房间（胜利条件为Score）并邀请选手进入参赛，请至少提前5分钟进房，准时开赛

  5. 未能及时到场的选手将全部做弃权处理，若剩余人数少于4人，则跳过淘汰赛阶段全部晋级

  6. 淘汰赛图池将在积分赛结束之后放出

决赛阶段：

  1. 比赛模式： BO9，9局5胜，由选手1V1对决，交叉选图

  2. 时间：将在淘汰赛结束后公布

  3. 比赛房间：裁判将提前15分钟建立Team VS模式的房间（胜利条件为Score）并邀请选手进入参赛，请至少提前5分钟进房，准时开赛

  4. 未能及时到场的选手将做弃权处理

  5. 决赛人数不足4人时，自动空出名次

  6. 决赛图池将在淘汰赛结束之后放出

淘汰赛和决赛的具体细则会根据情况作出适当的更改，请选手们密切留意
